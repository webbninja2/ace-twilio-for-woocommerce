== Description ==

This plugin help us to send sms when user placed order, It update order status to user.

###⚡️ FEATURES

> * Simple using shortcodes
> * Set the texts for date, time
> * Customize boxes color
> * Customize of colors for text and background

== Installation ==

1. Go to the Plugins Menu in WordPress
2. Search for **" IMS Countdown "**
3. Click **" Install "**
4. Click **" Active "**
5. Now you can see Ace Twilio menu in sidebar. Where you can add Countdown for your page or post.

== Screenshots ==
